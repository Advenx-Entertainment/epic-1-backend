CO SYSTEM — CLIENT SETUP
=========================

This guide is for setting up device_client.py on remote PCs / Raspberry Pi.
Run this on every device that connects to the arena server.


REQUIREMENTS
------------
Python 3.8+

Install websockets:
    pip install websockets
or
    pip3 install websockets


STEP 1 — COPY FILE
------------------
Copy device_client.py to the remote PC.
Place it anywhere, e.g. Documents/Websockets/device_client.py


STEP 2 — EDIT device_client.py
--------------------------------
Open device_client.py and change these two lines at the top:

    DEVICE_ID  = "pc-remote-01"
    SERVER_URL = "wss://postdoctoral-acorned-terrance.ngrok-free.dev/ws/pc-remote-01"

DEVICE_ID:
    Give each device a unique name.
    No spaces, use hyphens.
    Examples: rpi-laser-01, pc-node-02, arena-chip-03

SERVER_URL:
    Replace the URL with the one from the ngrok terminal on the main PC.
    The URL looks like: https://postdoctoral-acorned-terrance.ngrok-free.dev
    Change https:// to wss:// and add /ws/YOUR_DEVICE_ID at the end.

    Example:
    ngrok gives you:  https://postdoctoral-acorned-terrance.ngrok-free.dev
    SERVER_URL becomes: wss://postdoctoral-acorned-terrance.ngrok-free.dev/ws/pc-remote-01

    NOTE: The ngrok URL changes every time ngrok is restarted on the main PC.
          Update SERVER_URL here whenever that happens and rerun the script.


STEP 3 — RUN
------------
    python device_client.py
or
    python3 device_client.py

Expected output:
    [pc-remote-01] Connecting to wss://postdoctoral-acorned-terrance.ngrok-free.dev/ws/pc-remote-01
    [pc-remote-01] Connected.

If you see Connected, the device is live and receiving commands from the server.


STEP 4 — VERIFY ON MAIN PC
---------------------------
On the main PC, open the browser at:
    https://postdoctoral-acorned-terrance.ngrok-free.dev

The log will show:
    Received: {"type": "status", "event": "device_connected", "device_id": "pc-remote-01" ...}

This confirms the remote device is connected.


ACCESSING THE DASHBOARD FROM REMOTE PC
---------------------------------------
Open browser on the remote PC and go to:
    https://postdoctoral-acorned-terrance.ngrok-free.dev

In the Server URL field enter:
    wss://postdoctoral-acorned-terrance.ngrok-free.dev

Enter a Device ID and click Connect.
Messages sent from any connected PC will appear in all browsers in real time.


MULTIPLE DEVICES
----------------
Repeat Steps 1 to 3 on each device.
Each device must have a unique DEVICE_ID.

Example for 3 devices:
    Device 1:  DEVICE_ID = "rpi-laser-01"
    Device 2:  DEVICE_ID = "rpi-sensor-02"
    Device 3:  DEVICE_ID = "pc-node-03"

All use the same SERVER_URL.


SENDING COMMANDS
----------------
To send to all devices:
    {"type":"command","action":"laser_trip","target":"all"}

To send to one specific device:
    {"type":"command","action":"laser_trip","target":"rpi-laser-01"}


COMMON ERRORS
-------------
HTTP 404                   -> SERVER_URL is wrong or server is not running on main PC
server rejected connection -> ngrok is not running on main PC, restart it
NEWURL in SERVER_URL       -> you forgot to replace the placeholder with the actual URL
python not found           -> use python3 instead of python
pip not found              -> use pip3 instead of pip
connected but no messages  -> make sure browser on main PC is also using the ngrok URL not localhost
